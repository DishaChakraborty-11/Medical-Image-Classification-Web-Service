# Dataset

This project uses the Brain MRI Images dataset from Kaggle:

https://www.kaggle.com/datasets/navoneel/brain-mri-images-for-brain-tumor-detection

Due to GitHub size limits, the dataset is not included in this repository.

## Download Instructions

1. Install Kaggle CLI
2. Place your kaggle.json in C:\Users\<your-username>\.kaggle
3. Run:

kaggle datasets download -d navoneel/brain-mri-images-for-brain-tumor-detection
unzip brain-mri-images-for-brain-tumor-detection.zip
4. Place the extracted folders inside `data/`
